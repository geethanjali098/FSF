Variables in CSS:(sep29)

Variables in CSS, also known as custom properties, allow you to store and reuse values throughout your stylesheets. They are defined using the -- prefix followed by a name and a value. Here's how you declare a CSS variable:


:root {
--main-color: #007bff;
--font-size: 16px;
}
:root selector is used to define variables at the root level (global scope) of your stylesheet.
You can use these variables throughout your CSS rules by referencing them with the var() function. For example:
body {
background-color: var(--main-color);
font-size: var(--font-size);
}
h1 {
color: var(--main-color);
}
Using variables makes it easy to update styles across your entire website by changing the variable value in one place.

Media Queries in CSS:
Media queries in CSS allow you to apply different styles based on the characteristics of the user's device or viewport, such as screen size, resolution, or device orientation. Media queries are typically used for creating responsive web designs. Here's the basic syntax for a media query:
@media media-type and (media-feature) {
/* CSS rules to apply when the media query conditions are met */
}

media-type can be screen, print, speech, etc., specifying the type of media the query applies to.
media-feature defines the condition under which the styles should apply, such as max-width, min-width, orientation, and more.
Here's an example of a media query that changes the background color of a page when the viewport width is less than 600 pixels:

@media screen and (max-width: 600px) {
body {
background-color: lightgray;
}
}

You can also combine multiple media features and types to create complex queries. For instance, this media query applies styles when the viewport width is less than 600 pixels and the device is in portrait mode:
@media screen and (max-width: 600px) and (orientation: portrait) {
/* Styles for small screens in portrait mode */
}
